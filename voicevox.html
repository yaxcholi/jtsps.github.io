Side Projects

(1) Voicevox

Project Page

VOICEVOX is an open source, deep learning, reading/text-to-speech synthesizer Japanese software developed by Hiho.

Website Page

This project focuses on making a Text To Speech software using the voices provided by VOICEVOX, while the software itself is already a text-to-speech synthesizer I wanted to make more user-friendly applications. The software provided by VOICEVOX is not used for people to use with their microphones, rather it is mainly made for video editing and music videos hence the name VOICEVOX. This side project adds user features such as being able to select their microphone, play around with the sound, pitch, volume and even allows them to choose from speakers/voices.
